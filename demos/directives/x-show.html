x-show Directive

What is x-show?

x-show toggles element visibility using CSS display property. The element remains in the DOM but is hidden/shown. Unlike x-if, it doesn't add/remove elements.

Example 1: Simple Toggle

This content can be toggled on and off!

Current state:

Explanation: x-show="isVisible" shows the element when isVisible is true, hides it when false. The element stays in the DOM.

Example 2: Conditional Messages

✓ Operation completed successfully!
⚠ Please review your input!
✗ An error occurred!
Explanation: Multiple elements can use x-show with different conditions. Only one message shows at a time based on messageType.

Example 3: Form Validation

Sign Up Form

✗ Please enter a valid email address

✗ Password must be at least 8 characters

✓ All fields are valid!

Explanation: x-show can use methods and complex expressions. Error messages only appear after clicking "Validate" and when fields are invalid.

Example 4: Loading States

Loading... Please wait.

Data Loaded:

Click the button to load data

Explanation: x-show manages different UI states: loading, loaded with data, and initial state. Each state shows different content.
